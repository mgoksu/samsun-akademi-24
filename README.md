# Samsun Akademi 2024 Sunum

## Kurulum

1. Repoyu indirin.
   ```bash
   git clone https://github.com/mgoksu/samsun-akademi-24
   ```

2. requirements.txt dosyasından gerekli paketleri yükleyin.
   ```bash
   pip install -r requirements.txt
   ```
